來一場 Linux Mint 21.1 與 Manjaro 22.1.0 之間的 PK

發表於 2023-05-14 分類於 linux
文章字數： 2.5k 所需閱讀時間 ≈ 5 分鐘

前言

Linux Mint 與 Manjaro 在 DistroWatch 上，分別佔據第 3、4 名的位置，而且，第 3 名與第 4 名之間的差距還有點大。

最近在 Virtualbox 安裝與使用上述 2 支 Distro 一小段時間，因為，我拿到一部 HP ProDesk 600 G1 DM，便想說把測試過程印象良好的 Linux Mint 安裝上去把玩一陣子看看，玩著玩著就玩出今天要分享的這一篇小小心得。

在 VirtualBox 上的表現

  1. Linux Mint 21.1：表現穩定又輕快，而且系統預載的程式，剛好都符合我一進入系統可能會先安裝的那幾支。

  2. Manjaro 22.1.0：在 VirtualBox 上的體驗，我試過 Manjaro KDE 與 XFCE，只要開啟 EFI，安裝完之後，初次進入系統，我做完更新之後，就無法再進入系統。不要核取 EFI 還能夠用大絕招關機法，再重新開啟後，進入系統。由於 Manjaro 22.1.0 是最近才釋出更新的 Distro，不得不說這樣的表現，我是真的感到滿意外的。

在 HP ProDesk 600 上的表現

Linux Mint 21.1

  1. 原本，在 VirtualBox 上感受節的穩定與輕快，在 ProDesk 600 幾乎是感受不到的，特別是在安裝過程，那個彩色小球就轉很久。初始安裝完成後，我巡一下預設安裝的套件，發現來到實體機器上，並沒有被更動。或許，會有伙伴問我，這有什麼好比的，…說真的，也是因為 Manjaro 安裝到實體機器後，有些小地方確實與 VirtualBox 上安裝後呈現的不太一樣，因此，我才特別提出來說一下下。

  2. 接下來要分享的是我在 VirtualBox 上使用時沒有發現的問題，這個問題比較偏向是輸入法框架這方面的。那就是 Linux Mint 開始選單的搜尋功能，似乎與 Fcitx5 感情不好，我無論是用嘸蝦米還是新酷音，都無法正常輸入；換到 Manjaro KDE 開始選單的搜尋功能，Fcitx5 是沒有問題的。

  3. 整個系統在我設定好之後，想說泡杯咖啡來看一下有土伯上面的影片，想不到竟然能夠看影片看到系統掛點，發生的狀況，要不是整個畫面黑掉，就是系統自動閃退，重新回到登入畫面，這可以說是我近年來使用 Linux 從未見過的場面。第 1 支影片還沒有看完，重新登入系統再開啟 Firefox 後，Firefox 建議我重新開機，並重新指定 Profile 檔，…想不到按下重新開機後，系統跑滿久才又回到桌面，我當下沒想到要截圖下來，因為我認為是一個小小意外。但，…第 2 支影片，雖然沒有要我重新開機，又再掛點一次，我就開始轉向 Manjaro。

Manjaro 22.1.0

  1. 我決定安裝 KDE 版的 Manjaro 到 ProDesk 600 上頭，安裝期間，安裝程式突然閃退過 1 次(據我了解，有伙伴在網路上分享自己安裝 Manjaro KDE 22.0.5 到實體機器的經驗更不順利)，閃退後我沒有重新開機，就再執行一次安裝程式，也就「順利地(?)」把系統 On 上去。

  2. 必須說，系統運行不能算是輕快，但穩定度夠好。只是原本在 VirtualBox 可以輕鬆搞定的 Fcitx5，來到實體機器上，初始安裝完是看不到 Fcitx5 的，需要一步一步慢慢處理，回想起來，在 VirtualBox 與實體機上明顯的差異，就是實體機器上的安裝程式有過一次閃退，我不確定是不是跟這個事件有關。

  3. 我把 Fcitx5 與新酷音、嘸蝦米等安裝與環境設定簡述如下。 (1)先下 sudo pacman -S fcitx5 fcitx5-im fcitx5-chinese-addons fcitx5-configtool fcitx5-chewing fcitx5-table-extra (2)sudo nano /etc/enviroment，並將以下資料寫入，存檔後重新登入系統(或重新開機)。

    (3)用 Fcitx5 的圖形介面，把想要用的輸入法新增進來，原則上就搞定輸入法。

  4. XFCE 版預設將 gufw 安裝進來，KDE 版的話，一如在 VirtualBox 那般是要自行安裝的。

  5. 上面提到 Linux Mint 開始選單，使用搜尋功能時，Fcitx5 無用武之地，而 Manjaro KDE 則是正常發揮，OK 的。

無關 PK 的其他狀況

Linux 暫時與 NBA 無緣

  1. 應該是 2022 年 10 月開始的，就是…無論是那一支 Linux Distro，最明顯無解的狀況就是…，無法看 NBA 上面的影片。

  2. 有些 Distro 跟 Linux Mint 一樣可以看到個幾秒廣告，但不久後就像是被封禁一樣。

  3. 有些 Distro 跟 Manjaro 一樣，直接就是來一波像被封禁那般…。

  4. 我想到的解法，就是找到其他能夠觀看的系統裝置，再不就是用 VirtualBox 去安裝 Windows。…想不到網路上有人確實就是用 VirtualBox + Windows 來處理，我認為會這樣處理的人，要不就是手上只有那麼一部電腦，但，更大部分的可能性是再去找其他裝置還比較麻煩，乾脆弄個虛擬機器還比較快所致。

Line 沒有原生的程式

  1. 我不是很清楚大家怎麼處理這個問題，我的話，處理方式是安裝 Google Chrome + Line 的 Extension(傳送門：LINE - Chrome 線上應用程式商店)。

  2. 我認為，上述方式就是應應急，或是當下有需要才會用上。大部分的時間裡，我會直接把手機拿出來，或是切到 macOS、Windows 使用功能完整且原生的 Line 程式。
