Linux Mint 22 Beta

發表於 2024-07-25 分類於 linux
文章字數： 2.6k 所需閱讀時間 ≈ 5 分鐘

前言

在 DistroWatch.com 上，我注意到 Linux Mint 爬上「網頁參觀排名」第 2 名，並且在這個名次應該維持至少 3 個月以上。我認為這支 Distro 會是不少入門，並且有使用 Linux 桌面系統需求的人，很好的選擇。

從官方說明來看 Linux Mint 22

本篇文章，主要是參考官方文件：New Features in Linux Mint 22 ‘Wilma’ - Linux Mint，從而試著窺探版本號碼 22 的 Linux Mint，有什麼樣的新功能。

Better language support

  1. 先不管官方的說明，我在 Beta 版身上，安裝過程時，發現有些語言還沒有正常顯示，詳見下圖。

  2. 相較之下，Linux Mint 21.3 就正常多了。

  3. 從下圖來看，我猜 Linux Mint 22 可能是還在 Beta 階段的關係，所以，主要以英語系的語言為主。

  4. 雖然對其他語言的支援似乎還不足，但就我自己對 Linux 幾年下來的觀察，能夠將中文(包括簡體與繁體，通常以簡體為優先)支援，就率先放到 Beta 階段的，應該是少數。這一點，實在要感謝 Linux Mint 團隊。

Support for new technology

※官方說明列出 8 點，我自己注意到的是其中 2 點。

  1. Linux Mint 22 ships with modern components and the new Ubuntu 24.04 package base.

  2. The default sound server switched to Pipewire.

Support for popular features

官方列出 6 點，有興趣的伙伴可以自己閱讀一下。

Software Manager

  1. 這次的更新，說到「Unverified Flatpaks are disabled by default.」

  2. 只要打開之後，就可以「方便地」安裝到不少「Unverified Flatpaks」，方便很多。

  3. 但要注意的是…風險自負。

Matrix

  1. 以往，有一些問題時，透過 Google 爬文的話，有時候都會挖出論壇的文章，所以有些 Distro 都會自己建構自己的專屬社群論壇，方便使用者去找資料，…但，以我個人而言，我並沒有使用過這樣的資源。Linux Mint 想透過 Matrix 以 Real-time internet chat 的模式，讓使用者可以運用類似前述的資源，我認為這是一個好的嘗試。

  2. 對我來說，不論是論壇或是 Matrix，主要的問題比較是要再註冊一個帳號，這一點不是很喜歡，我也不喜歡有些服務要跟我原有的 Google 等等之類的帳號連動進行註冊。所以，就目前來說，這個功能的立意良好，但，我會有比較多的時間是多觀望。

Cinnamon 6.2

  1. 官方列出不少功能，我對這這些功能的看法是它們比較多是在細節上的打磨，而 Cinnamon 真的滿好用的，整個介面對大多數的 Windows 使用者而言，應該滿直覺地就能夠上手。所以，細節上的更新，對大多數的人而言，可能不是那麼要緊。這一點，我認為適用於大部分的 GUI 使用者。

  2. 然而，對於可能會長時間與 Linux Mint 泡在一起的人而言，倒是要花一點時間瞭解更新上的細節，例如：「Cornerbar applet: shift+click action added」這樣的功能，是不是對自己而言，帶來更多的便利等。

Other improvements

  1. 這裡應該是列出一些比較無法歸類，或是主題化的更新與改良。

  2. 我特別喜歡「In Timeshift, deleting a snapshot now pops up a confirmation dialog.」這個功能，因為，Delete 的時候，沒有再做一次 Confirmation，還滿容易遇到手殘把重要的 Snapshot 砍掉的狀況。

Artwork improvements

  1. 以前，我只用預設桌面。

  2. 這次 Linux Mint 22 Beta 特別提到 Artwork，我便換了張桌布感受一下。

  3. 是說，Ocean 一上來，整個人的心情很是平靜。

Main components

  1. 官方一句話帶出所有重點：「Linux Mint 22 features a Linux kernel 6.8 and an Ubuntu 24.04 package base.」

  2. 由於我自己很喜歡 ubuntu 24.04 LTS，Linux Mint 22 是立基在這個版本上的衍生發行版，加上 Linux Mint 在桌面使用與設計上的用心，我相信會往更好的方向發展。

LTS strategy

  1. 官方說明的意思，應該是直到 2026 年，應該都可以妥妥地使用 Linux Mint 22，而且 Linux Mint 22 將會持續安全性更新至 2029 年。

  2. 能夠進行 LTS(Long Term Support) 策略的主要原因，應該是就是立基於 ubuntu 24.04 LTS 吧！？

中文輸入法

  1. 推薦的輸入法框架：Fcitx5。

  2. 嘸蝦米輸入法：已經將 fcitx5-table-extra 納入軟體庫，所以，安裝上方便很多。

  3. 下面幾道指令就可以把「Fcitx5 + 嘸蝦米輸入法 + 新酷音輸入法」，一併安裝進來。

  4. 重新啟動系統之後，「新酷音」已經是預設的中文輸入法，再另外從圖形介面把與「嘸蝦米」加進來，就搞定。

  5. 附上安裝的影片，供大家參考。
