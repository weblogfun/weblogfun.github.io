Fedora 9. Make Waves.

發表於 2008-05-14 分類於 linux
文章字數： 1.7k 所需閱讀時間 ≈ 3 分鐘

Fedora 9. Make Waves. 我喜歡這樣的說法… (( 簡體中文的頁面，翻譯為引領潮流，我喜歡用興風作浪，感覺比較囂張。 ))

在下載的過程中，我以往的作法是去魔獸世界閒晃或是在網頁之間盡情衝浪 (( 因為英文通常會用 Surf 來表達網路瀏覽行為，又逢 Fedora 9 的 Slogan 跟浪花有關，所以我就亂搭一番，更詳細的字彙解譯，可參考Yahoo!奇摩字典，網址為 http://tw.dictionary.yahoo.com/search?ei=UTF-8&p=surf )) ，而今則是先去讀一下 Released Notes 。

以下是我試著將 Release Highlights 所提及的特性部分整理出來 (( 不要傳達錯誤訊息是我在寫資訊相關部落格時的基本原則，但也礙於能力有限，若仍有錯誤之處，煩請告知，我會馬上修正。 )) ：

  1. GNOME 2.22：GNOME 中新加入的套裝軟體 Cheese 可以利用網路攝像頭，創作照片和視頻。其他改進包括，改進的網路檔系統支援，新的國際化時鐘小程式， Evolution 中對 Google 日曆和自定義標籤的支持，新的遠端桌面查看器，增強的輔助特性，以及對 PolicyKit 的整合。
  2. KDE 4.0.3：據了解有可良好的整合與更優質的美術設計，讓人眼睛為之一亮。
  3. Xfce 4.4.2：坦白說，我只安裝過乙次，對它了解不深，只知道相較於 GNOME 與 KDE ，它可以在效能較差的機器上運轉自如。
  4. NetworkManager 0.7：將提供更好的 Wireless 支援，包括 GSM 和 CDMA 設備的支援，以及對多設備和特別網路共用連接的支援。NetworkManager 自此在被預設為啟用狀態，不用再下指令或是另外啟用之。
  5. Anaconda：Fedora 的安裝程式，將直接支援 ext2/3、NTFS 檔案系統，並以 FirstAidKit 改進 Rescue Mode ，重新設計過的 netboot.iso 也因此變得比較大隻點。
  6. Live USB 映象檔：現在支援永久性的改變，也就是說在使用過程中動到的設定與資料的讀出及寫入等，在重新開機後將會保留著，而不是跳回預設值。
  7. PackageKit：新的管理套裝軟體的圖形和命令行工具，並且提供跨平臺軟體管理的支援。在本次發行中，Pirut 被其替換，更新程式的部分將 Pup 替換。在背景作業的 Yum 性能也大幅提高。
  8. FreeIPA：提供更簡便的 Web 介面的伺服器管理方式，提供一個功能強大的 Fedora 目錄伺服器、FreeRaDIUS、MIT Kerberos、NTP 和 DNS 整合性解決方案。
  9. Ext4：Ext3 的後起之秀，擁有較高效能、較大容量的存取能力等令人振奮的特性。
  10. Upstart：將取代 /sbin/init 這支老 Daemon (( 更詳盡的部分可先參考 upstart - event-based init daemon )) 。
  11. Firefox 3：新外觀，其原本功能中的 Bookmarks 也被 Places 取而代之。
  12. OpenJDK 6：完全自由開放的 OpenJDK 6 成為預設安裝的 Java 運作環境，IcedTea 7 不再是預設安裝的套件，GCJ 仍是 PowerPC 架構下的預設套件。
  13. OpenOffice.org 2.4：應不用多做說明，它將是 Fedora 9 重要的一部分。
  14. Perl 5.10.0：佔用更少的記憶體，在其他部分也有進步。
  15. TexLive：取代已不再被維持、更新的 TeX 套件。
  16. Kernel 2.6.25：Fedora 9 的核心。
  17. Kerneloops：Kernel 掛點，將會被友善的 Kerneloops 自動回報至 http://www.kerneloops.org/ 。
  18. X：開關機的程序將有明顯的效能改善。

以上，整理的過程中，我已經決定以 Live USB 跟隨 Fedora 9 興風作浪 (( 語畢，下載 Fedora-9-i686-Live.iso 的進度已經來到 49%。 )) 。
