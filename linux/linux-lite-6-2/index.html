來玩 Linux Lite 6.2

發表於 2023-01-30 分類於 linux
文章字數： 1.4k 所需閱讀時間 ≈ 2 分鐘

前言

今天要分享的 Linux Distro 是 Linux Lite，目前在 DistroWatch 排名在第 8 名，而我，在安裝這支 Distro 的過程中，因緣際會地讓實體機器上原本已經使用一段時間的 Garuda Linux，被 Linux Lite 所取代。

快速回顧與感想

  1. 第一印象：同樣地，我使用 VirtualBox 搭配 ISO 檔進入系統，第一印象就是配置上，…跟 Windows 10 以前的視窗桌面好像(下圖)，雖然，我也遇過其他 Distro 做到很接近的，但 Linux Lite 帶給我一種更貼近的感覺，然而，心裡頭仍舊非常清楚，這的確就是一支 Linux。

  2. 在 VirtualBox 上的預設解析度：最近幾支 Distro 的桌面環境，在 VirtualBox 裡的表現都不是很理想，但，Linux Lite 就比較可圈可點。因為，Linux Lite 自行抓到一個比較好的顯示結果，表現出來沒有什麼問題。

  3. 安裝過程：我在安裝時，觀察到一個狀況：安裝程式(Installer)的效能不太優秀，本以為是在 VirtualBox 才這樣，我為了進一步求證，轉而安裝在實體機器上，想不到…，結果是一樣的。關於這一點，可以點擊下列影片，這支影片是我在 VirtualBox 上所錄製的，我另外觀察到一點，就是實體機器上，安裝步驟來到「更新和其他軟體」時，其項下的「安裝 Linux 同時下載更新」是無法被核取的。

  4. 決定讓 Linux Lite 在實體機器上 Long Stay：那麼，明明安裝過程就是有點慢，也因此誤打誤撞地安裝在實體的機器上，我還是決定讓這支 Distro 走一趟 Long Stay 呢？
    (1)第一個答案是效能，我發現安裝完成後，系統在執行上，效能都算良好，也沒有明顯的穩定度問題。
    (2)這一支 Distro 是基於 Debian 與 ubuntu 所開發的，與 Linux Lite 相關的資料多少可以推斷，Linux Lite 的整合度夠好，才能夠從 2012 年持續維護與發展至今。
    (3)我想要感受一下：Linux Lite is a ‘gateway operating system’. It was created to make the transition from Windows to a linux based operating system, as smooth as possible.
