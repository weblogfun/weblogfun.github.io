來玩 elementary OS 7.1

發表於 2023-10-08 分類於 linux
文章字數： 662 所需閱讀時間 ≈ 1 分鐘

前言

elementary OS 7.1 (Horus) 釋出後幾天，我決定也來安裝看看。

語系設定

  1. 上次我安裝 elementary OS 來玩，已經是 6.0 版的事，當時的 6.0 版在語系設定上有點狀況，這個部分在 7.1 已不復見 (或許更早之前就處理好，只是我直到現在才又安裝，把玩時發現的)。
  2. elementaryOS 設定語系的過程有點不太一樣，先是在安裝時選用語系，安裝完成後，初次進入系統時會再做一次語系選擇與設定，讓使用者在後續的使用上決定以什麼語系為主。
  3. 一進到這個畫面，還會有一段語音包(雖然沒有仔細聽，但應該是用「英文」製作而成的)。
  4. 安裝過程，選擇 Taiwan 所顯示的簡體中文，這並不影響安裝及後續的語系選擇，所以，請放心使用。
  5. 這是因為，安裝完成選擇語系時，就會正常發揮。

安裝嘸蝦米輸入法

  1. fcitx(4)：直接在 Terminal 下一道指令：sudo apt install boshiamy-table-boshiamy，接著重新登入系統，就搞定。
    (↓)參考影片
  2. iBus：系統自動預選的輸入法框架，安裝的方式，可以直接參考行易有限公司官網說明。
  3. Fcitx5：一樣沒有收錄 fcitx5-table-extra，所以，想要快速使用的話，我會比較建議用 fcitx(4) 就好。

後記

  1. 本次安裝使用的 ISO 檔版本是 elementaryos-7.1-stable.20230926rc.iso。
  2. elementary OS 7.1 Available Now ⋅ elementary Blog
