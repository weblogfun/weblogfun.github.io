發表於 2023-04-22 分類於 linux
文章字數： 656 所需閱讀時間 ≈ 1 分鐘

前言

幾天前，我才剛在 VirtualBox 裝上 Manjaro Linux 22.0.5 Plasma Edition，想不到，沒幾天就迎來一個升級版更新。

呃…，那就趁這個時候先試著更新一波，同時，看看能不能順利升級。

反正，頂多就是把一部虛擬主機搞爛而已。

升級流程

  1. 我想，先把幾個指令直接寫下來(如下)。

  2. 下列影片是下完指令後的升級過程紀錄，影片最後，可以看到我再下一次 lsb_release -a 這道指令確認是否成功升級。

後記

  1. 我相信，這個升級過程，基本上沒有什麼難度。更新所花費的時間約 15 分鐘，這個速度僅供參考，主要還是依個人所使用的電腦性能、網路速度等等，而會有差異。
  2. 我在撰寫本篇貼文的時候，無聊地注意到 Plasma Edition 上 Manjaro 的 Logo 已經走圓角化，但 XFCE Edition 好像還是跟之前一樣：沒有圓角。
  3. 至於 Manjaro 22.1.0 有什麼樣的更新，有興趣的人可以參考這個傳送門的說明：Manjaro 22.1 Talos released - Announcements / Releases - Manjaro Linux Forum。
