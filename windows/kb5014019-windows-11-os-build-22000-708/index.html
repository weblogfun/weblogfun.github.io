[版本更新] 快速回顧 微軟發布 KB5014019 (Windows 11 作業系統組建 22000.708)

發表於 2022-05-27 分類於 windows
文章字數： 1.6k 所需閱讀時間 ≈ 3 分鐘

前言

5 月 24 日，微軟針對 Windows 11 釋出 KB5014019 (作業系統組建 22000.708) 的更新。

更新前內容概覽

  1. 官方說明文件：May 24, 2022—KB5014019 (OS Build 22000.708) Preview，截至本篇紀錄發稿前，官方的說明還沒有繁體中文版。另外，說明文件提到這支更新的性質(如下)。

    This non-security update includes quality improvements. Key changes…

  2. 上面的網址是「https://support.microsoft.com/en-us/topic/…」，有興趣的伙伴，可以試著把「en-us」改成「zh-tw」或「zh-cn」，再按「Enter」鍵。我試著改成「zh-tw」後，目前得到的結果如下圖，就是…說明文件還是維持英文，但網頁導覽等其他地方，呈現的是繁體中文。

  3. (2022 年 5 月 31 日更新說明)我今天試著用上述方法重新進入頁面，發現已經轉為繁體中文的說明頁面(如下圖)，相信這個語言還是相對親切許多。

  4. 從上一支補丁(May 10, 2022—KB5013943 (OS Build 22000.675))至今，官方的說明都還沒有中文化，可能是有些時候中文化之後，還要讓使用者「中翻中」，所以，我猜這些說明，有可能暫時不會再做中文化，我是滿期待未來有更完美的中文化說明後，再次展現給使用者閱讀。(2022 年 5 月 31 日這天，我再試著進入，發現已有繁體中文說明。)

  5. 這一次，我會更新的原因，主要是想要測試一下「Windows 焦點」(Windows spotlight)，細節的部分，我留到下一段再做說明。

    Windows spotlight on the desktop brings the world to your desktop with new background pictures. With this feature, new pictures will automatically appear as your desktop background. This feature already exists for the lock screen.

  6. 其實，另外一個更新：Microsoft Family Safety，我個人也滿期待的，只不過，我家小朋友的作業系統雖然都可以更新到 Windows 11，但，我目前仍然保守地讓機器們都停在 Windows 10。是也滿期待 Windows 10 也有類似的更新與改善，這個部分，我再找時間去確認一下下。

    Improves the Family Safety verification experience for a child account when you send a request for additional screen time.

「Windows 焦點」(Windows spotlight)

我之前都用 Bing Wallpaper 來達到持續變換桌布的效果，透過這支 App 更換桌布，會讓我產生一種從窗內望向窗外無界限的感覺。

而今次更新裡的「Windows 焦點」，感覺比較多是把類似的功能，直接整合到系統裡面，使用者就不用另外安裝 App。

不過，如果您跟我一樣，也有安裝 Bing Wallpaper 的話，就會發現這 2 個功能並不是協同合作的。

大致上的狀況是「Windows 焦點」改動了 Bing Wallpaper 設定的桌布，系統就會丟出下列的通知。

如果我們啟用 Bing Wallpaper，「Windows 焦點」會從設定裡面自動跳掉。

後記

我正在思考，是不是要繼續使用 Bing Wallpaper。

另外，我另外一部機器早就跑完更新，也拿來處理工作上的事，度過一整天(還包括加班)。

截至目前在使用上，沒有特別奇怪的狀況發生。
