後記

  1. 本次安裝使用的 ISO 檔版本是 archlinux-2021.08.01-x86_64.iso。
  2. 儘管，我老早就知道 Arch Linux，但，這是我第一次安裝 Arch Linux，遲遲沒有安裝是因為我想用桌面環境，只想在幾個步驟，就讓安裝程式把所有的東西都安裝好，但是，隨著近日嘗試過幾支 Distro 之後，我反而對於太自動就安裝好的，不一定都能夠接受。只能說，自動有自動的好處，手動也有手動的優勢，而 Arch Linux 就是一個幾乎都要手動的 Linux，…就我所知，應該僅次於 Gentoo Linux 吧！
  3. 以一個新手而言，如果想要對 Linux 建立起較紮實的觀念，從 Arch Linux 真的可以學到很多。
