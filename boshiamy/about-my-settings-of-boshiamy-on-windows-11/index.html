我在 Windows 11 上面設定嘸蝦米的筆記

發表於 2022-02-10 分類於 boshiamy
文章字數： 1.2k 所需閱讀時間 ≈ 2 分鐘

前言

這一篇貼文，主要是紀錄我在 Windows 11 上面做一些與嘸蝦米輸入法相關設定的流程，我將設定大致分為 2 個方向：

  1. 設定嘸蝦米為預設且唯一的中文輸入法
  2. 保留其他中文輸入法，設定嘸蝦米為「預設輸入法」

※本篇貼文內，使用的嘸蝦米版本是「嘸蝦米 J」。(2022 年 9 月 20 日更新)

設定嘸蝦米為預設且唯一的中文輸入法

  1. (↓)【開始】→【設定】，跳出以下畫面。

  2. (↓)點左側的【時間與語言】，再點該項目下的【輸入】。

  3. (↓)接著，再點【輸入】項下的【語言與地區】，並且進到【語言與地區】的設定細項。

  4. (↓)點【中文(繁體，台灣)】右邊的【．．．】(就是【點點點】)，跳出並點擊【…語言選項】。

  5. (↓)或許，大家有注意到，安裝完嘸蝦米輸入法之後，並不會出現在【已安裝的鍵盤】裡。

  6. (↓)沒有關係，點【新增鍵盤】，把【嘸蝦米輸入法】加進來。

  7. (↓)加完嘸蝦米輸入法，因為我只留嘸蝦米輸入法，所以，就移除掉【微軟注音輸入法】。

  8. (↓)回到桌面上的工作列右下角，找到【英】或【中】，點右鍵跳出嘸蝦米輸入法的設定選單，然後點選單上的【設定】。

  9. (↓)這一部電腦，我想要一進系統以【英數模式】為預設輸入模式，並且不要【繼承系統輸入法狀態】，也不要【使用 Shift 切換「中文/英文」模式】。

保留其他中文輸入法，設定嘸蝦米為「預設輸入法」

  1. (↓)在 Windows 11 上面，排在「第一個」的輸入法並不是所謂的「預設輸入法」，所以，下圖可見嘸蝦米輸入法好像在排序上是「第一個」，卻不會因此成為「預設輸入法」。

  2. 以微軟注音輸入法與嘸蝦米輸入法共存為例，原則上有下列 2 種做法。
    (↓)其一，【設定】→【時間與語言】→【進階鍵盤設定】→【覆寫預設輸入法】(也可以在【設定】介面裡的【尋找設定】，直接用關鍵字把設定找出來)，調整為嘸蝦米輸入法。
    (↓)其二，進入【微軟注音輸入法】的設定，把【預設輸入模式】改為【英數字元】，這樣的做法是讓登入系統時以「英數字元」為預設值，需要切換到嘸蝦米輸入法時，就用 Win+Space 做切換。

後記

  1. 與 Windows 10 預設為嘸蝦米輸入法之設定方式 相比，就可以發現嘸蝦米輸入法本身的設定是一樣的，不同的是【設定】介面的部分， Windows 11 與 Windows 10 有所不同。
  2. 題外話…，隨著不少新的電腦已搭載 Windows 11 出貨，我相信目前的 Windows 11 已經有一定的穩定度，而我此次更新到 Windows 11 後，會持續觀注系統補丁，而且不一定每一個補丁釋出，都立馬跟著更新，主要是希望，這一回升級到 Windows 11 至少要能夠穩穩用個 1 年。
