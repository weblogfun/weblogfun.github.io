解決重新安裝 macOS High Sierra 時，卡關在「無法聯絡復原伺服器。」(2024 年 7 月 17 日更新)

發表於 2024-03-17 分類於 apple
文章字數： 2.4k 所需閱讀時間 ≈ 4 分鐘

前言

今天，我想把 macOS High Sierra 砍掉重練，所以，重開機之後，就按住 Command + R，準備來一場重灌之旅。

想不到，天不從人願，在我執行 macOS High Sierra 的安裝程式時，跳出「無法聯絡復原伺服器。」

卡關，無法繼續，Google 之後，發現不少人遇到這樣的問題。

我整理一下，總共有 4 種解法，並且紀錄在本篇貼文裡。

解法(一)：改變 NVRAM 軟體更新網址

  1. 說明：開啟 Terminal，先確認一下所使用的系統時間是否正確，再更改 NVRAM 軟體更新網址。

  2. 這個解法，我個人沒有試過，主要是我得知這個方法後，思考一下就決定這一次不走這個方法。原因是 http 並沒有加密，這樣的安裝，我認為有安全性相關的疑慮。至於，系統上寫入的 NVRAM 軟體更新網址本來是不是走 https，以我個人淺薄的經驗是無法去進一步考究的，只知道現在能夠走不加密的 http，是有機會成功復原到 macOS High Sierra。

解法(二)：「可開機安裝磁碟」安裝法

  1. 說明：拿出可開機安裝磁碟(通常使用 USB 隨身碟製作)，重開機按住 C，進入可開機安裝磁碟就能夠迎刃而解。

  2. 我所使用的是 SD 卡做為可開機安裝磁碟，然而我的 Mac Mini（2014 年末） 上面的 SD 讀卡機怪怪的，即使「偶爾」能夠讀得到 SD 卡，也無法順利從 SD 卡上開機。我也有想到 USB 的 SD 讀卡機，因為少用的關係，老早就送出去。又或者，先從其他 Mac 主機製作可開機安裝磁碟，就有機會搞定，但，Mac Mini（2014 年末）已是我現階段僅有的 Mac 主機，思來想去的結果，就是這個方法暫時是行不通的。

解法(三)：網路復原安裝法

  1. 說明：重開機，立即按住 Option + Command + R，更新到比較新的 macOS。

  2. 以 Mac Mini（2014 年末） 這部機器而言，最終支援的系統版本是 macOS Monterey，我之前曾把系統升級到較新版的 macOS Mojave 與 macOS Big Sur，無奈硬體性能幾乎都扛不住後續版本的系統，因此，映入眼簾的 macOS Monterey 著實讓我有點掙扎，也因此讓我想到後面會提到的「組合技解決法」。

解法(四)：組合技解決法

  1. 說明：先走「網路復原安裝法」，如果系統運行實在太慢，就在新系統裡製作可開機安裝磁碟，走「『可開機安裝磁碟』安裝法」，回到 macOS High Sierra。

  2. 有一點可能要注意：macOS Monterey 通常會是在 Apple 檔案系統（Apple File System，簡稱 APFS）上運行，然而，我記得 macOS High Sierra 雖然也支援 APFS，但是它的系統安裝程式似乎無法直接將 macOS High Sierra 安裝在 APFS 上頭。印象中，我的經驗是安裝到一半時就會卡住不動，如果您也遇到這樣的狀況，或是想要避免這樣的狀況，建議可以在重新安裝 macOS High Sierra 時，將硬碟格式化為 Mac OS 擴充格式（日誌式），應該就可順利服用。抑或是堅持想讓 macOS High Sierra 在 APFS 上頭運行的話，我會建議從更早之前的版本，安裝後再升級到 macOS High Sierra，若我沒有記錯，升級的過程就會詢問使用者要不要改用 APFS。

後記

  1. 目前手邊僅存的 Mac Mini（2014 年末），曾經，被拿來安裝過幾支不同的 Linux Distro。但，我得說，好玩歸好玩，除非 macOS 都跑不動，不然，原汁原味的 macOS 還是比較合口味。

  2. 原本系統預測要 2 小時 19 分鐘才能夠完成安裝 macOS Monterey，實際上不到 1 小時就能搞定，時間花費還是有點久(我的網路環境是「台灣大寬頻 1G」)。升級到 macOS Monterey 後，我感受到比較多的是整合上帶來的方便，其中一項是我在 iOS 上購買的幾支 App，現在也能夠在 macOS Monterey 使用，這感覺超爽的，問題就是系統運行不到 1 個月，就跑得實在不是普通的慢。

  3. 我讓 macOS Monterey 運行 4 個月，還是決定用一支狀況良好的隨身碟製作成「可開機安裝磁碟」，讓自己回到 macOS High Sierra，降刷的時間約半小時就可以搞定。這次降刷，除了嘸蝦米輸入法外，只安裝 Firefox 與 Facebook Messenger 這 2 支 App。說實在的，回到 macOS High Sierra 的感覺真好。

  4. 降刷讓老機器價值催到底，至今還可以持續使用 macOS High Sierra，算是非常夠本。
